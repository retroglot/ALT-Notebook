# Ⅰ自败的理论

## 一、间接自败的理论

**自败的理论**　有的理论是即使以其自身的标准来衡量也是失败的，从而宣告自己完全无用。这种理论不需要假设。而许多最有名的理论在某些方面都是自败的。


### 1. 自利的理论

**自利论的目标**　理论的目标，分为形式上的目标与实质性的目标，帕菲特特指“实质性的目标”。而自利论 S 给每个人设立了一个目标：对他自己来说结果要最好，而且会使他的生活过得对他自己而言尽可能地好。

**对自利论的解答**　在应用 自利论 S 的情况下，所有目标中有三种可能性最大：

- *快乐论*
- *愿望实现论*
- *目标列表理论*

**快乐论**　　对某个人来说最好的东西就是会带给他最大幸福的东西。

**愿望实现论**　对某个人来说最好的东西就是一生中最满足他愿望的东西。

**目标列表理论**　即便我们不想拥有好的事物或不想规避坏的事物，某些事物对我们来说也仍然有好有坏。

**自利论的主要主张**　在目标分类的前提下，即一些目标是终极目标，一些目标是工具性目标的情况下，我们可以重新表述自利论的主要主张：

> （S1）人人都有一个至高无上的合理的终极目标：使自己的生活过得对自己而言尽可能地好。


### 2. 自利论何以可能是间接自败的

**T设目标**　假设某个理论为 T ，该理论给我们设定的目标为我们的 *T设目标* 。

**间接自败的**　当某人努力去实现 *T设目标* 时，这些目标的达成总体上将更糟，则称 T 的理论是间接地、个体地自败的。

间接自败不简单判断 T 是否自相矛盾，而是讨论 T 在某个时间段应用于某些人时是否会自败。

**S设目标的自败定义**　如果我力图做任何对我而言最好的事情，对我而言反而都将变得更糟。

当出现两类情形时，会达到这个特别的定义：

```
(a) 我经常因为自利而做出适得其反的后果，比如我以为偷东西谋生不会被抓，我就为了让自己更好而经常偷东西，导致我被抓受罚，最终我没有达到让自己更好的结果。
```

(a)类情形源自我试图遵循 S 的纯粹自利时，由于我判断失误或能力不足导致我做出了对我而言更糟的举动。

反驳者认为这是我的缺陷，而不是 S 的缺陷，因为在 S 的情况下我需要执行偷东西不被抓，而我由于无能被抓到了，并且我忽视了诚实可能是对我而言最好的策略。

因此，(a)类情形的“自败”是一种实践层面的失败，不算对 S 的反驳。它没有证明 S 作为理论是无效或自我矛盾的。

```
(b) （在我可能做出的那些行动中，即便我从不做那种对于我而言将会更糟的事，但是）如果我是纯粹自利的，那么对于我而言也可能是更糟的。因为如果我具有某种别的意向的话，对我而言或许更好。
```

(b)类情形指向我完美执行的“纯粹自利”导致了我的实践出现了适得其反的结果。

有一种情况，明知两种方式中没有哪一种对我来说更好，而我还是按照其中某个方式行动。或者，我哪怕选择了对我来说更好的方式，但我是在按照更为特定的愿望而非该目的行动。比如我给爱人送礼物，结果得到了爱人的忠诚，但我并不是为了纯粹自利而去做送礼物的事，而是出于爱本身。
