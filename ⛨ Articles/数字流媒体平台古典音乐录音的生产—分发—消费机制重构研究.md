# 数字流媒体平台古典音乐录音的生产—分发—消费机制重构研究

# 引言

## 研究背景

进入21世纪，以数字化与网络化为核心的技术革命，对全球音乐产业的底层架构进行了颠覆性重构。数字流媒体服务的崛起与普及驱动着从介质形态、产业结构到商业逻辑的全方位重塑。

工业时代唱片业的价值链是线性的，每一个环节的准入壁垒高，信息与权力高度集中于以唱片公司为代表的少数头部机构，这一生态在分发端受到了来自于网络数字音乐巨大的冲击。流媒体音乐平台的出现重组了消费生态，它凭借技术优势，将内容聚合、版权管理、数据分析、分发推广与用户交互等功能集于一身，通过在消费端的低价垄断，逐渐成为了连接产业链上下游所有参与者的绝对中枢。这种平台化的生态系统极大地降低了音乐分发的门槛，使得独立音乐人与小型公司得以绕过传统唱片公司的繁琐流程直面听众，同时也使唱片公司的角色从“生产者”与“发行者”向“版权服务与增值管理者”加速转型。

“内容创作　→　投资制作　→　复制发行　→　渠道销售　→　终端消费”（图）

最为关键的是，在商业逻辑层面，产业的核心价值锚点从产品计价转向了流量变现。　传统模式下，商业闭环的实现依赖于专辑或单曲的单位销售，而在流媒体时代，以“订阅制”和“广告支持下的免费服务”为代表的商业模型，将价值评估的重心转移到了注意力经济的范畴。在此框架下，用户的收听时长、互动行为、分享意愿等共同构成了可被量化的“流量”。其中，“播放量”这一核心数据指标，被赋予了前所未有的多重功能属性：它既是衡量作品市场热度的价值标尺，也是版权收益按比例分配的结算依据，更是驱动平台算法推荐系统进行个性化内容分发的决策引擎。这一系列变革共同塑造了一个以数据为生产要素、以流量为运营导向、以算法为分发机制、追求即时用户反馈与规模化传播效应的全新产业环境。

古典音乐从录音产业诞生之初就成为其附属消费的重要一环，而古典音乐录音的商品市场在渠道消费中不断被其他新兴音乐风格挤占，最终成为较难获得收益的一环。网络数字音乐的发展则进一步加剧了对古典音乐录音的冲击。“网络时代对传统唱片的冲击，造成目标受众群体规模被大幅度缩小”【1】　。这种边缘化在产业结构上体现得更加尖锐。以国内为例，2024年的唱片产业市场规模约11.98亿元【2－报告】，尽管受到网络数字音乐的影响市场规模不稳定波动，也仍属于较大的总体产业市场，然而，古典音乐细分领域却呈现出萎靡态势。“乐团发行唱片的数目在不断减少，整个市场表现出萎靡的状态。”【3】古典音乐资深从业人员表示不愿意录制唱片，而是更愿意进行演出形式的现场表演【根据采访】，认为唱片销售基本不属于其表演收益来源。

特定的古典音乐录音在实体唱片主导的时代，始终有部分固定听众收藏，其生产端策划逻辑与音乐会类似，通常以乐章为分轨，将整部作品录制为单个或半个唱盘。进入数字音乐主导的第二次变革后，生产端更倾向于以单曲作为单位在网络上进行公开分发，即使有时保留了传统唱片的生产习惯，也以每单曲作为分轨，将大约十首单曲打包作为合集录制为一个唱盘。这种生产端、分发端的变化深度影响了消费者的消费习惯，直接导致短时长歌曲逐渐成为音乐最小消费单位的消费模式。这与古典音乐录音基于单次完整作品而非单曲的消费逻辑、较长的聆听时长，与算法驱动的碎片化、场景化消费模式形成了更深层的结构性矛盾。最终，古典音乐在大众传播看似触达更广泛潜在听众的同时，却在以播放量、用户时长为核心的价值链体系中被进一步稀释，稳固在长尾末端，成为一种“广泛可及却收益微薄”的文化存在。

数字音乐产业发展的当下，行业实践多集中于单一环节的优化，如版权保护技术的升级或个别平台的垂直化尝试，却缺乏对“生产—分发—消费”价值链的视角观察。不仅是音乐表演从业者个体，表演艺术院团、院校与音乐公司都难以找到自身发展路径的方向。因此，深入剖析数字流媒体平台如何重构古典音乐录音的产业机制，识别各环节中的核心矛盾与利益冲突，并进行具体实践，成为了至关重要的任务。本研究旨在探讨第一步，即指出古典音乐录音产业的价值链机制内部各环节的具体矛盾。

## 研究问题

数字流媒体平台正在逐渐成为古典音乐录音传播的核心渠道之一，这一过程深刻重构了传统唱片工业的生产、分发与消费机制。由于古典音乐艺术属性的特殊性，通用音乐产业的数字化逻辑在此领域的具体影响尚未可知，与此同时，产业链各核心主体面临着独特的转型困境与利益博弈。由此本文提出以下的研究问题：

1. 古典音乐录音产业链各环节的核心主体面临哪些转型困境与利益冲突？

2. 流媒体平台通过何种方式重构了古典音乐录音的传播机制？

3. 未来应如何设计一种数字生态系统，以有效调和古典音乐的艺术完整性、深度聆听体验与流媒体经济的规模化、流量化运营之间的根本矛盾，实现其文化的可持续传播？

## 研究对象

本文讨论的研究对象是古典音乐的录音产品背后的“生产—分发—消费”机制。古典音乐录音产业属于整体音乐产业下，录音产业的细分部分。古典音乐录音产品在两个渠道进行分销，一是通过传统录音产业分销渠道进行实体专辑的制作与线上或线下销售，二是通过流媒体音乐平台进行以专辑或整场演出录音回放为单位的上架。本文聚焦于第二种渠道，即数字流媒体平台作为新渠道，重组并深刻影响传统“生产—分发—消费”机制的过程。

本文所讨论的古典音乐录音产品，主要指以西方艺术音乐传统为核心，包括传统，由专业乐团、室内乐团体或独奏家演绎，以作品演绎或音乐会实况为内容的录音，但也包括其他传统音乐形式如民乐、当代打击乐等专业领域，可以理解为流行音乐的补集。

### 传统录音产业的策展式音乐生产机制

- 策展式生产机制的定义与内核
	- 传统的录音
- 决策逻辑：内容导向与专家驱动
	- 内容导向，指的是生产者在产出音乐的时候会有一个主题目标，然后通过主题目标去寻找相应的音乐进行录制
	- 专家驱动，指的是由非常少数的一批所谓的“策展人”在做着这样的事情。这是起源于音乐会的变体，并发展于电台时代，由点扩散到面的。在这里可以论述两段话：音乐会本质是一种策展；电台的专家已经把歌曲都选好了，从而形成了消费习惯，消费习惯反作用于生产。
- 专辑作为音乐的核心产品形态与价值载体
- 商业模式与专辑形态的绑定
	- 到了后来，专辑形态就成为了音乐消费的一种固定模式
	- 举例：人们在谈论“音乐”时往往会联想到这是什么专辑的音乐；或者在评价音乐的时候往往讨论谁出了“新专”。

### 流量导向的反作用：专辑生产模式的消解

- 专辑逻辑某种意义上的失败
	- 人们原生性的就会把歌曲和歌手绑定，而专辑模式增加了一道中介，规训了人们。但最终这样的后天规训走向了失败。
	- 热门单曲作为专辑主打歌，反向异化了专辑逻辑，这导致人们往往只为了一两首歌去买专辑
	- 精选热门专辑“大拼盘”也是类似的逻辑，这导致人们跳过了专辑，直接去找歌手+热门歌曲
- 收听习惯从专辑到单曲与歌单的重塑
- 平台架构设计与收益模式加速专辑解体
- 古典音乐录音价值依赖完整性的受损

### “音乐家有限收入”困境

- 这是一个新概念，这个概念的意思是：
	- 生产40分钟的“一套”古典音乐包装成产品，比生产3分钟的“一首”流行音乐并包装成产品要来的困难？（好像有点站不住脚）
	- 音乐家从录音制品中获得的收入与其付出的艺术劳动、承载的文化资本严重不匹配的现象。
		- 为什么？古典领域下，音乐家演奏的曲目范围是有限的，而每一首曲子最终都会被最头部的表演家进行演绎，从分配的角度上说，生产“土耳其进行曲”这种名曲的背后是“我的产品”与“顶尖艺术家30年前的成功产品”的竞争
		- 这是一种时空的错位竞争
	- 曲目的消费者盘子本来不大，不仅被不同时空下的历史录音抢占收入，还被同时空下的头部顶尖音乐家抢占收入
- 参考的理论内容：
	- **Baumol's Cost Disease（鲍莫尔成本病）**：表演艺术无法通过技术提高生产率，成本却持续上升
	- **Winner-takes-all Market（赢者通吃市场）**：少数明星获得大部分收入，大多数人收入微薄
	- **Precarious Labor（不稳定劳动）**：零工经济下的职业不稳定性
- 录音产业与现场演出行业共生关系的断裂
- 价值分配权力转移导致音乐家收益被稀释

当前，中国音乐产业正在经历一场由全球数字流媒体驱动的大变革。音乐的传播介质从实体转向数字虚拟，分销模式从传统渠道转向平台化生态，服务销售从实体销售转向了免费增值的模式，于是，线下体验和复制品区域分销所建立的音乐商业模式，正在悄然被数字平台的新生态重构，而这种重塑是不可逆的。信息、符号与资本突破了地理界限和时间约束，形成了以数据流为核心的“流动空间”(Castells)，这种空间逻辑重构了文化生产的时空维度，使音乐从地域性产品转变为全球性流动符号。互联网的存在现在创造了一个全球化的民主化信息环境，导致对高质量和低质量信息的平等获取(Bickley,2025)。平台依靠算法搭建起“注意力经济”为核心的数字生态，在这种新生态下，播放量成为新指标，由此，传统的价值分配机制被围绕单曲形态构建的流量化传播逻辑所取代。

古典音乐有其独特的生产惯例，在实体录音复制品层面，生产链通常以“专辑”作为媒介，可将其视为“策展式”(Garofalo,1999)的生产链条，旨在通过完整的作品编排或精选集复刻音乐会的审美体验。这一垂直领域通常偏居一隅，处于小众的产品生态位，在生产链中很大程度上保留传统惯例。然而，音乐数字化引导的转型打破了这一惯例。流量作为新的价值评估参考系，使传统的 LP 专辑模式逐渐走向衰落，市场的注意力被海量的短 EP 、单曲和歌单蚕食。而古典音乐的曲目平均时长、曲式结构等元素在新的数字场域中天然处于劣势，难以被公域流量曝光或私域分享，导致其深厚的文化资本无法有效转化为经济资本。

此外，摇滚乐、电子乐等流行音乐产业中的垂直领域与古典音乐的垂直领域之间形成了高度区隔、相互独立的艺术场域(Bourdieu)。两种场域在音乐产品平台化的背景下演化为两个相互独立的价值网络。生产端的计算机制作和乐器实录、分发端的歌曲品牌化和歌曲编目化、消费端的粉丝经济和精英主义，都在独立发展，在整个生产链上缺乏跨域流动性。这种分化的根源，来自于不同场域的惯习与资本形态存在不可通约性，最终将前述的音乐产品艺术价值与经济价值的矛盾固化为两个平行场域间的结构性壁垒。

在中国，这种结构性矛盾尤为明显。以音乐厅为代表的剧场音乐消费惯习在改革开放之前未能在中国建立历史根基与辐射范围；除了实体 CD 售卖之外，千禧年之际的免费盗版网络音乐下载填补了群众音乐消费需求的空白。这共同塑造了中国用户对“免费+搜索”模式的强大的路径依赖。这种路径依赖迫使古典音乐产品作为音乐产品的子集同样只能嵌入国内流行音乐产业以流量为核心，以网络互动为维持方式的变现体系，这放大了播放量、互动量等数据指标的权重，削弱了艺术价值在音乐传播中的地位。检索结果碎片化、元数据与版权信息不规范或缺失，使曲目检索本身变得更加艰难，进一步加剧了古典音乐同曲目不同录音版本的分流。这些因素使古典音乐在中国数字音乐产业生态中的边缘化处境愈发严峻。

现有研究多聚焦于流行音乐的数字化转型，对古典音乐这一特殊领域在数字平台时代的生存策略研究不足。特别是在国家大力推动文化数字化战略，《“十四五”文化发展规划》提出“推动传统艺术形式与现代传播方式深度融合”的政策背景下，探讨古典音乐的数字化转型问题尤为迫切。中国的音乐产业环境中，古典音乐如何在维持艺术完整性的前提下实现文化价值与商业价值的良性循环，成为亟待解决的应用问题。
