# 科尼利尔斯·范德比尔特

>   19 世纪，美国 **商业基础设施** 的奠基者与垄断者。

### 主要贡献

- 他建立和整合的航运与铁路系统，是美国19世纪经济快速发展的关键基础设施，极大地促进了全国范围内的人员、商品和原材料的流通。

- 他通过兼并和收购来创建大型企业集团。这种运营方式为后来的美国大型股份制公司和托拉斯的发展提供了范例。

- 通过其商业帝国，他积累了巨额财富，成为当时世界上最富有的人之一。晚年捐赠了100万美元，用于创建范德比尔特大学。

### 标志性事件

| 战略            | 具体表现                                            | 案例分析                                         |
| :------------ | :---------------------------------------------- | :------------------------------------------- |
| **价格战之王**     | 低于成本定价 → 逼对手破产 → 收购 → 垄断后提价                     | 在与哈德逊河航运垄断权的斗争中，将票价降至1美分并免费提供食物，迫使对手让步并收购公司。 |
| **垂直整合与网络效应** | 他不满足于只控制运输环节，而是通过收购铁路、煤矿、轮船公司，构建了从源头到终端的完整运输网络。 | 控制纽约中央铁路并连接其他线路，形成无法绕行的运输网络。                 |
| **资本运作大师**    | 擅长利用股权、债券和华尔街的金融工具进行兼并重组，以小博大。                  | 通过股票操纵和市场压力，获得湖岸与密歇根南方铁路控制权。                 |
| **结果导向管理**    | 他的管理风格极其直接，要求下属绝对服从，只看结果，不问过程。运营效率被压榨到极致。       | 改革铁路公司：裁员、统一标准、优化时刻表，提升运营效率和利润。              |
