# **循道宗复兴运动**
**Methodist Revival**

> 18世纪，由 [[约翰·卫斯理]] 及其兄弟查尔斯·卫斯理领导的基督教复兴运动，起源于英国，是 [[福音派运动]] 的核心组成部分。该运动强调个人皈依、圣经权威、社会正义和积极福音传播，不仅重塑了英国宗教格局，还对全球基督教发展和社会改革产生了深远影响。

### 基本信息

**核心教义**
- 以 **“心灵宗教”** 为中心，强调个人与上帝的亲密关系和内在信仰体验：
- 通过悔改和信仰追求“第二次祝福”（完全成圣），以实现属灵生命的更新；
- 坚持圣经为最高权威，倡导简单、直接的信仰实践，反对形式主义；
- 注重敬虔生活与社会行动的结合，推动慈善、教育、医疗和废除奴隶制等事业；
- 它并非旨在创立新的教派，而是意图在英国国教圣公会内部进行革新，重振信徒的个人信仰与道德生活；
- 采用创新的组织模式，如巡回布道、班级会议和年会制度，以维持社区活性和信仰传播。

**历史地位**
- 被公认为 **现代福音派运动的奠基性事件**，为19世纪以来的全球宗教复兴浪潮提供了范本。
- 确立了循道宗作为世界最大新教教派之一的基础，在英美、非洲及亚洲等地拥有广泛影响力，至今仍是基督教领域的重要分支。

**贡献**
- 引领了大规模的 **宗教复兴**，在工业化和社会动荡时期重振了基督教信仰的活力，吸引了大量底层和中产阶级信徒；
- 催生了关键 **社会改革运动**，直接推动了废奴主义（如通过威廉·威伯福斯等追随者）、监狱改革和公共教育普及；
- 开发了高效的 **教会组织与传教模式**（如巡回牧师体系、平信徒领导），为现代跨文化传教和教会增长策略树立了典范；
- 促进了 **全球基督教网络的形成**，通过传教活动将循道宗传播至美洲、非洲和亚洲，深刻影响了当地宗教与文化发展。
