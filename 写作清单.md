
| 框架模块        | 核心任务：回答什么问题？                     | 在您的研究中可以这样写（示例）                                                                                                                    |
| :---------- | :------------------------------- | :--------------------------------------------------------------------------------------------------------------------------------- |
| **1. 研究背景** | 故事的开端：这个领域发生了什么宏大变化，以至于值得我们关注？   | 简要阐述数字流媒体如何成为音乐消费的主流模式，同时点出古典音乐在这一浪潮中遭遇的普遍困境和业界焦虑。**（点到为止，制造悬念，不超过一页）**                                                            |
| **2. 研究问题** | 聚焦核心：基于上述背景，我具体要研究哪个“点”？         | **（这是全文的“心脏”）** 将我们之前提炼的核心问题和子问题，一字不差地、清晰地罗列在此处。例如：“基于上述背景，本研究旨在回答以下核心问题：……”                                                       |
| **3. 文献综述** | 前人种树：关于我的问题，学界已经有哪些讨论？还存在什么空白？   | 分类梳理：a) 数字音乐平台商业模式的研究；b) 古典音乐产业转型的研究；c) 媒介仪式理论等相关理论的研究。最后明确指出：“**现有研究……但鲜有研究将三者结合，系统性地剖析古典音乐在数字分发中的结构性矛盾。**” 这就凸显了您的研究空白和价值。       |
| **4. 研究意义** | 价值宣告：回答了我的问题，能带来什么贡献？            | **理论意义：** 将“注意力经济”、“仪式理论”等跨学科理论引入艺术管理研究，为理解数字时代的艺术生产与消费提供新视角。**实践意义：** 为古典音乐厂牌、艺术家、数字平台等从业者提供可参考的突围策略。                             |
| **5. 研究对象** | 靶心在哪：我具体分析的案例/数据/现象是什么？          | 明确您的研究将聚焦于主流流媒体平台（如Spotify, Apple Music）与新兴古典音乐垂直平台（如IDAGIO, Primephonic）上的古典音乐内容。研究对象包括但不限于：平台的产品形态、算法机制、版权分配模式、代表性古典音乐厂牌的数字化策略等。 |
| **6. 研究方法** | 用什么工具：我打算如何收集信息、如何分析，来回答我的问题？    | 可能会用到：**文献研究法**（梳理理论）、**案例分析法**（深入剖析特定平台或厂牌）、**比较研究法**（对比流行音乐与古典音乐、主流平台与垂直平台）、**话语分析法**（分析乐评、用户评论等文本）。                             |
| **7. 文章框架** | 路线图：我的论文将分为几章，每章干什么，彼此之间是什么逻辑关系？ | 将我上次为您设计的“问题驱动型”大纲放在这里。每一章的标题都应清晰地反映其核心内容和在整个论证链条中的位置。                                                                             |
